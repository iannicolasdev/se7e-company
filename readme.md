# Catálogo Interativo de Produtos - Se7e Company

Catálogo digital desenvolvido sob medida para a loja **Se7e Company**, com foco em praticidade, navegação por categorias e design responsivo voltado para dispositivos móveis.

## Funcionalidades
- Filtro de produtos por categoria com scroll no dropdown
- Botão "Visualizar mais / menos"
- Modal com mais informações ao clicar em um produto
- Botão de voltar ao topo
- Design responsivo para dispositivos móveis

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Como usar
Clone o repositório:

```bash
git clone https://github.com/seu-usuario/se7e-company.git
```
Abra o arquivo index.html no seu navegador.

Ou, se preferir, [acesse a versão online aqui](https://se7e-catalogo.netlify.app/)

## Inspiração
Projeto criado como uma solução prática e visual para catálogos de produtos, com foco em acessibilidade, experiência do usuário e boa apresentação em dispositivos móveis.
